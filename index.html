eip2612pa

Static auditor for EIP-2612 permit implementations. Point it at a Pharos contract and it walks the deployed bytecode, finds the permit() entry, and runs 8 correctness checks against the EIP-2612 spec — selector presence, nonces, DOMAIN_SEPARATOR, type-hash literal, ecrecover, nonce increment, deadline, owner check.

// how it works

  1. fetch bytecode via eth_getCode
  2. extract the 82-byte Permit(...) type-hash (PUSH-N aware)
  3. find the permit() selector (0xd505accf)
  4. walk opcodes near it: STATICCALL to 0x01, TIMESTAMP, EQ, SLOAD→SSTORE
  5. render: PASS / FAIL / WARN / NOT_FOUND per check
