# eip2612pa-demo

Live browser demo of [eip2612pa](https://github.com/jamiuke/EIP-2612PA) — an EIP-2612 Permit Auditor for the Pharos Network.

**Live site:** https://jamiuke.github.io/eip2612pa-demo/

## What this repo is

A single static `index.html` that runs the same 8-check audit logic as the eip2612pa Python/bash scripts — but in the browser. It calls the live Pharos RPC via CORS and renders the audit as a terminal log (green-on-black, monospace).

## Source

The skill itself lives at https://github.com/jamiuke/EIP-2612PA.
